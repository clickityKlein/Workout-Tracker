Instructions


Welcome to Workout Tracker, a system designed around analytics and flexibility! Workout Tracker allows a user to be as specific as they'd like in tracking their data and includes the option to download csv files of workout data.


Getting Started

  1. Create an Account (through sidebar or homepage)
  2. Set Recovery Questions (through account page)
  3. Start Logging Workouts!

Creating Cardio Workouts:

  1. Select Create Cardio Workout (through sidebar or homepage)
  2. Fill out the First Form:
    • Workout Name: can be generic or specific, however you would like to track your data
    • Date: there is a calendar widget to aid in selection, don't worry about not logging a workout the day of
    • Location: can be generic or specific, however you would like to track your data
    • Number of Exercises:
      • Just went for a run? Enter 1!
      • Training for a triathlon? Entering 3 will allow you to track 3 separate cardio exercises for a single workout (i.e. did you run, cycle, and swim?)!
    • Number of Circuits:
      • Just went for a run? Enter 1!
      • Doing some circuit training? Suppose you did 3 intervals of a certain exercise, enter 3.
  3. Log Exercises and Circuits in the Second Form:
    • Exercise: can be generic or specific, however you would like to track your data
      • Automatically populated with exercise numbering, but this can be overwritten.
      • Back to the triathlon example, if you specified 3, you could name the exercises "Run", "Swim", and "Bike".
    • Split: time of the given exercise and circuit.
    • Distance: how far did you go?
    • Metric: how did you track the distance?
      • Can choose either imperial, metric, or other for non-conformative styles of training
    • Circuit: automatically populated, can be overwritten to best represent your workout
    • Order in Circuit: automatically populated, can be overwritten to best represent your workout
    • Notes: anything you'd like to mention for tracking your workouts!

Creating Strength Workouts:

  1. Select Create Strength Workout (through sidebar or homepage)
  2. Fill out the First Form:
    • Workout Name: can be generic or specific, however you would like to track your data
    • Date: there is a calendar widget to aid in selection, don't worry abuot not logging a workout the day of
    • Location: can be generic or specific, however you would like to track your data
    • Number of Exercises:
      • Just want to log some pushups? Enter 1!
      • Did you do pushups, pullups, and situps? Entering 3 will allow you to track 3 separate strength exercises for a single workout!
    • Number of Sets:
      • Did you do 3 separate sets of bench press? Specify 3!
  3. Log Exercises and Sets in the Second Form:
    • Exercise: can be generic or specific, however you would like to track your data
      • Automatically populated with exercise numbering, but this can be overwritten.
      • Can specify exercises like "pushups", "squats", "benchpress", etc. here.
    • Weight: Did you do curls with 20lbs? Enter 20.
      • Can leave at 0 for body weight.
    • Metric: choose between body weight, imperial, or metric.
    • Repetitions: how many reps did you do during this set?
    • Set Number: automatically populated, can be overwritten to best represent your workout
    • Order in Set: automatically populated, can be overwritten to best represent your workout
    • Notes: anything you'd like to mention for tracking your workouts!

Data:

  • Using either the homepage or sidebar, Detailed and Summarized data can be accessed for your account
    • Detailed: every exercise from all sets/circuits
    • Summarized: line item for each workout
    • Single Workout View: Navigate through the tables to zero-in on a single workout view. If this is your workout, you'll have the option to edit or delete this workout.
    • Downloads are available for detailed and summarized data on the homepage.
  • Using either the homepage or sidebar, navigate to the Aggregate Reports to view data across all users
    • Detailed: every exercice from all sets/circuits
    • Summarized: line item for each workout
    • Single Workout View: Navigate through the tables to zero-in on a single workout view. If this is your workout, you'll have the option to edit or delete this workout.
    • Downloads are available for detailed data on the aggregate page.
